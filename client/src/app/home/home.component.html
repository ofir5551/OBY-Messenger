Messenger

This web application was developed by Ofir Ben Yamin.

Since it is hosted on a Heroku Server, the entire data will be wiped every few hours as there is no persistent database.

In order to begin sending and receiving messages, please sign up here.

Compose - Send messages to other users by their IDs.

Manage - View sent and received messages that belong to the currently logged user.

Users List - This is basically an address book. It lists all registered users and their IDs.

Login - Login to an existing user to begin using the application.

Signup - Register as a new user.

Page not found - This page will kindly redirect you here in case you get lost :)
